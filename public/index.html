Timestamp API

A simple API that takes either a natural language date (ie January 15, 2017), or unix time code as a GET request and responds with a JSON object containing both the natural and unix represenation of that date.

If an invalid request is made, responds with the same JSON object with Null as the values.

Natural Input: url/january%2015%202015
(note: %20 is how browsers interpret a space character).

Unix Time-Code Input: url/1421244000000.
